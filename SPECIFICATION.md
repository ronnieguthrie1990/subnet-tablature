This manual discloses an instructional framework, structural schema, and 

architectural method for shifting the visual and spatial representation of an 

instrument fingerboard from continuous sequential fret numbering to an organized 

\[Subnet, Index] coordinate design matrix, refactoring the physical layout 

into an extensible, segmented node network.



This document constitutes a formal, intentional defensive publication of the 

Subnet Tablature architecture. It establishes an immediate, open prior-art

structural chunking, and coordinate system described herein. This public 

disclosure ensures that the core architecture remains forever open, unencumbered,

The prior-art scope of this disclosure explicitly encompasses all variations 

of the 4-fret repeating subnet structure, the designation of localized gateway 

anchor markers, string-agnostic coordinate mapping, and any digital, mechanical, 

or visual implementations used to materialize this coordinate system on stringed 

musical instruments.

\-------------------------------------------------------------------

VISUAL NAVIGATION \& HAND ORIENTATION

\-------------------------------------------------------------------

The indices 1 through 4 do not dictate strict fingering assignments. 

Instead, they represent a relative spatial index within each 4-fret 

Subnet block. 



When navigating the fretboard, a player's hand naturally treats the 

Gateway Marker (★) as a tactile and visual "home pointer." By anchoring 

the index finger at this low point, the rest of the hand automatically 

aligns with the remaining frets in the chunk. The numbers 1, 2, 3, and 4 

simply identify the note's distance from that base anchor. 



Players are encouraged to use whatever natural fingering or ergonomic 

hand positions are most comfortable for physical execution.

HARDWARE \& SYSTEM SPECIFICATION

===================================================================

\[01] Open strings are globally designated as Index 0.

\[02] The root fret of each active Subnet block (Fret 01, Fret 05, 

&#x20;    Fret 09, Fret 13, Fret 17, Fret 21, etc.) serves as the 

&#x20;    primary visual or tactile Gateway boundary anchor.

\[03] Physical fretboard markers, materials, inlay shapes, or 

&#x20;    side-dot configurations are left entirely to the builder's 

&#x20;    discretion to fulfill this sub-netting layout.

\[04] For truncated necks (e.g., 21-fret), implement Subnets 1 

&#x20;    through 5 fully, using Fret 21 as the final standalone 

&#x20;    Gateway anchor point.
